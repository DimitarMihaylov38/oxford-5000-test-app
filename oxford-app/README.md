# 📘 Oxford 5000 Test App

Просто и ефективно приложение за тестване на английски речник, базирано на Oxford 3000™ и Oxford 5000™ word lists.

## 🚀 Функционалности

* Избор на ниво: A1 → C1
* Комбинирани нива:

  * A1–A2
  * A1–B1
  * A1–B2
  * ALL (всички нива)
* 100 въпроса на тест
* Резултат в проценти
* Преглед на грешните думи
* Повторно решаване на грешните
* Вграден речник със:

  * преводи
  * част на речта
  * ниво
* 🔍 Търсене в речника (по английска дума или български превод)
* ⚡ Бърз UX с визуален feedback (✔ / ✖)
* 🧠 Нормализация на отговорите (без значение от главни букви, интервали и вариации)

---

## 📊 Данни

Приложението използва:

* Oxford 3000™ и Oxford 5000™ (Oxford Learner's Dictionaries)
* Преводи чрез FreeDict (EN–BG)

Общо:
👉 4028 думи

---

## 🛠️ Технологии

* TypeScript
* Vanilla JavaScript (без framework)
* Vite
* Modular architecture (separated UI + logic)

---

## 🧠 Архитектура

Проектът е разделен на:

```txt
src/
  core/       → бизнес логика (test engine, words, normalize)
  state/      → app state
  ui/         → render функции (screens)
  data/       → JSON с думи
  main.ts     → controller / routing
```

* Накрая:

  * списък с грешките
